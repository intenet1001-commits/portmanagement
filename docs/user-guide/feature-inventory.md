# 기능 인벤토리

> 포트 관리 프로그램의 모든 기능을 초보자 관점에서 정리한 문서입니다.
> 각 기능은 "무엇을 하는지"와 "초보자가 찾기 어려운 정도"를 함께 표기합니다.

---

## 프로젝트 관리 탭

개발 중인 프로젝트들의 포트, 폴더, 실행 스크립트를 한 곳에서 관리하는 탭입니다.

### 상단 툴바 (최상위 버튼들)

#### 기능명: 내보내기 (Export)
포트 목록 전체를 JSON 파일로 저장합니다. 다른 컴퓨터로 이전하거나 백업할 때 사용합니다.
[초보자 발견 난이도: 쉬움]

#### 기능명: 불러오기 (Import)
JSON 파일에서 포트 목록을 복원합니다. 기존 목록에 병합됩니다.
[초보자 발견 난이도: 쉬움]

#### 기능명: Supabase Push (동기화 업로드)
현재 내 기기의 포트 데이터를 클라우드(Supabase)로 업로드합니다. 다른 기기에서 Pull로 받을 수 있습니다.
[초보자 발견 난이도: 보통]

#### 기능명: Supabase Pull (동기화 다운로드)
클라우드에서 내 기기 데이터를 받아옵니다. 앱 시작 시 자동으로 한 번 실행됩니다.
[초보자 발견 난이도: 보통]

#### 기능명: 설정 (Settings)
Supabase URL/Anon Key, 기기 이름, 디바이스 ID 등을 설정합니다. 최초 설정 마법사(SetupWizard)도 여기서 열 수 있습니다.
[초보자 발견 난이도: 쉬움]

#### 기능명: 세팅 (SetupGuide)
Supabase 테이블을 만드는 SQL을 Claude 프롬프트 형태로 복사할 수 있는 가이드입니다.
[초보자 발견 난이도: 어려움]

#### 기능명: 로그 복사
앱 내부에서 발생한 활동 로그를 클립보드로 복사합니다. 문제 해결 시 유용합니다.
[초보자 발견 난이도: 어려움]

### 빌드 버튼들 (Tauri 빌드용)

#### 기능명: 앱 빌드 (.app)
macOS .app 번들을 생성합니다. Applications 폴더에 설치 가능한 형태입니다.
[초보자 발견 난이도: 보통]

#### 기능명: DMG 빌드
배포용 DMG 파일을 생성합니다. 다른 사람에게 앱을 공유할 때 사용합니다.
[초보자 발견 난이도: 보통]

#### 기능명: Windows 빌드 (.exe)
Windows용 NSIS 설치 파일을 만듭니다. Windows 환경에서만 동작합니다.
[초보자 발견 난이도: 어려움]

#### 기능명: DMG 폴더 열기
빌드된 DMG가 저장된 폴더(`~/cargo-targets/portmanager/release/bundle/dmg/`)를 Finder에서 엽니다.
[초보자 발견 난이도: 보통]

#### 기능명: DMG 출시하기
빌드된 최신 DMG를 바탕화면(Desktop)으로 자동 복사합니다. 빠르게 배포할 때 사용합니다.
[초보자 발견 난이도: 보통]

### 작업 루트 (Workspace Roots)

프로젝트 폴더들이 모여있는 상위 디렉토리를 등록해두면, 그 아래 하위 프로젝트를 빠르게 추가할 수 있습니다.

#### 기능명: 작업 루트 추가
프로젝트들이 모여있는 상위 폴더를 등록합니다. (예: `~/Documents/GitHub`)
[초보자 발견 난이도: 보통]

#### 기능명: 작업 루트 편집/삭제
등록된 루트의 이름을 바꾸거나 제거합니다.
[초보자 발견 난이도: 보통]

#### 기능명: 작업 루트 하위에서 프로젝트 추가
루트 아래 하위 폴더를 선택하면 자동으로 프로젝트가 등록됩니다.
[초보자 발견 난이도: 어려움]

### 포트 추가 폼

#### 기능명: 포트 번호 입력
개발 서버가 사용하는 포트 번호(예: 3000, 9005)를 입력합니다. 선택 사항입니다.
[초보자 발견 난이도: 쉬움]

#### 기능명: 프로젝트 이름 입력
포트 목록에 표시될 이름을 입력합니다. 비워두면 AI가 자동으로 이름을 추천합니다.
[초보자 발견 난이도: 쉬움]

#### 기능명: 폴더 경로 선택
프로젝트 폴더의 절대 경로를 선택합니다. Tauri 모드에서는 네이티브 폴더 선택 다이얼로그가 열립니다.
[초보자 발견 난이도: 쉬움]

#### 기능명: .command 파일 선택
실행 스크립트(.command 파일)를 선택합니다. 드래그 앤 드롭도 지원됩니다.
[초보자 발견 난이도: 보통]

#### 기능명: 선택 입력 토글 (Optional Fields)
배포 URL, GitHub URL, 터미널 명령어 입력란은 기본적으로 숨겨져 있습니다. "추가 정보" 토글을 눌러 펼칠 수 있습니다.
[초보자 발견 난이도: 어려움]

#### 기능명: 배포 URL 입력
Vercel/Netlify 등 배포된 서비스의 URL을 저장합니다. 목록에서 원클릭으로 열 수 있습니다.
[초보자 발견 난이도: 보통]

#### 기능명: GitHub URL 입력
프로젝트의 GitHub 저장소 URL을 저장합니다.
[초보자 발견 난이도: 보통]

#### 기능명: 터미널 명령어 입력
Supabase push/pull 등 자주 쓰는 터미널 명령을 저장합니다. 원클릭 복사 가능.
[초보자 발견 난이도: 어려움]

3가지 기준으로 정렬할 수 있습니다.

#### 기능명: 오름차순/내림차순 토글
정렬 방향을 바꿉니다 (↑/↓ 아이콘).

#### 기능명: 즐겨찾기 필터 (별)

### 포트 목록 카드 (각 프로젝트 카드 내부 버튼)

#### 기능명: 실행 (▶)
.command 파일을 실행해 서버를 시작합니다. 성공 시 Toast 알림이 뜹니다.
[초보자 발견 난이도: 쉬움]

#### 기능명: 중지 (■)
실행 중인 프로세스를 종료합니다. SIGTERM → SIGKILL 순으로 안전하게 종료합니다.
[초보자 발견 난이도: 쉬움]

#### 기능명: 강제 재실행 (⟳)
좀비 프로세스나 정지가 안 되는 서버를 SIGKILL로 강제 종료 후 즉시 재실행합니다.
[초보자 발견 난이도: 보통]

#### 기능명: 로그 보기 (Terminal)
Terminal에서 `tail -f`로 실시간 로그를 확인합니다.
[초보자 발견 난이도: 보통]

#### 기능명: tmux 실행
tmux 세션을 만들어 Claude CLI와 함께 프로젝트를 엽니다.
[초보자 발견 난이도: 어려움]

#### 기능명: tmux 재연결 (tmux↺)
기존 tmux 세션을 재활용해 다시 연결합니다.
[초보자 발견 난이도: 어려움]

#### 기능명: Claude 열기
해당 프로젝트 폴더에서 Claude CLI를 실행합니다.
[초보자 발견 난이도: 어려움]

#### 기능명: Bypass Permissions 토글
Claude CLI 실행 시 `--dangerously-skip-permissions` 플래그를 추가할지 선택합니다.
[초보자 발견 난이도: 어려움]

#### 기능명: 폴더 열기

[초보자 발견 난이도: 쉬움]

#### 기능명: 배포 사이트 열기

[초보자 발견 난이도: 쉬움]

#### 기능명: GitHub 열기

[초보자 발견 난이도: 쉬움]

#### 기능명: 터미널 Push 실행
저장된 터미널 명령어를 Terminal에서 바로 실행합니다 (예: `git push`).

## 숨겨진 기능 TOP 10

초보자가 쉽게 찾지 못하는 강력한 기능들입니다. 아는 사람만 쓰는 꿀팁!

### 1. Cmd+F 검색 단축키
검색 버튼을 클릭하지 않아도 `Cmd+F`로 즉시 검색창을 열 수 있습니다. Esc로 닫힙니다.
[초보자 발견 난이도: 어려움]

### 2. 드래그 앤 드롭으로 .command 파일 추가
포트 추가 폼에 .command 파일을 끌어다 놓으면 포트 번호와 폴더 경로가 자동으로 채워집니다. 별도 도구 `포트에추가.command`도 지원.
[초보자 발견 난이도: 어려움]

### 3. 작업 루트(Workspace Roots) 등록
상위 폴더를 한 번 등록해두면 그 아래 모든 하위 프로젝트를 빠르게 추가할 수 있습니다.
[초보자 발견 난이도: 어려움]

### 4. Git Worktree 자동 포트 할당
메인 포트가 3000이면 worktree는 30001, 30002, ... 자동 할당. 여러 브랜치 동시 개발 가능.
[초보자 발견 난이도: 어려움]

### 5. AI 일괄 보강 (Batch Enrichment)
수십 개 포트의 이름/카테고리를 **단일 Claude 호출**로 몇 초 만에 처리합니다. 개별 호출 대비 10배 빠름.
[초보자 발견 난이도: 어려움]

### 6. 강제 재실행 (⟳) 버튼
중지가 안 되는 좀비 프로세스를 SIGKILL로 죽이고 바로 재실행합니다. "서버가 먹통"일 때 생명줄.
[초보자 발견 난이도: 어려움]

### 7. Bypass Permissions 토글
Claude CLI를 `--dangerously-skip-permissions` 플래그로 실행해 권한 확인 없이 바로 작업 가능 (고급 사용자용).
[초보자 발견 난이도: 어려움]

### 8. 터미널 명령어 저장 & 원클릭 실행
Supabase push/pull, git 배포 스크립트 등을 저장해두고 클릭 한 번에 Terminal에서 실행. 복사 버튼도 따로 있음.
[초보자 발견 난이도: 어려움]

### 9. 기기 전환 & 리매핑(Remapping) 모달
다른 기기에서 Pull한 뒤 폴더 경로가 달라도 리매핑 모달로 `/Users/A/...` → `/Users/B/...` 일괄 치환 가능.
[초보자 발견 난이도: 어려움]

### 10. tmux + Claude CLI 통합
tmux 세션을 자동 생성해 Claude CLI를 프로젝트 폴더에서 바로 실행. 여러 프로젝트를 병렬로 AI와 작업 가능.
[초보자 발견 난이도: 어려움]
